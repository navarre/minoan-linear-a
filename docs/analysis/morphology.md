# Minoan Morphology: A Grammar Sketch

## Based on 758 word forms from Younger's Lexicon (2024)

---

## The Suffix System

Linear A has **8 productive suffixes** that account for 27.7% of all words:

| Suffix | Count | Function | Key examples |
|--------|-------|----------|-------------|
| **-ja** | 42 | Adjectival/ethnic; marks **persons** | pa-se-ja, se-to-i-ja, su-ki-ri-te-i-ja |
| **-te** | 39 | Dative/locative | a-di-ki-te, ta-na-te, mi-nu-te |
| **-na** | 38 | Genitive/agentive (or article) | ki-re-ta-na, pa₃-ni-na |
| **-ra** | 34 | Agent/category; marks **things** | ku-ku-da-ra, pi-ta-ra |
| **-se** | 26 | Instrumental/ergative | pi-ta-ka-se, a-na-nu-si-ja-se |
| **-ne** | 22 | Dative plural? | da-ku-se-ne, qi-tu-ne |
| **-ro** | 16 | Resultative/totaling | ku-ro, ki-ro, sa-ro |
| **-me** | 11 | Possessive "my" | ja-sa-sa-ra-me, da-me |

## Suffix Stacking (Agglutination)

27 distinct two- and three-suffix combinations found. Suffixes stack in **fixed order**:

| Chain | Example | Parse |
|-------|---------|-------|
| **-ra-me** | ja-sa-sa-**ra-me** | thing + possessive = "my lady" |
| **-ra-me-ne** | ja-sa-sa-**ra-me-ne** | thing + possessive + plural/locative |
| **-na-te** | a-ta-**na-te**, ta-**na-te** | genitive + locative = "of/at X" |
| **-ja-se** | ki-ta-na-si-**ja-se** | ethnic + instrumental |
| **-ra-ja-se** | ja-pa-**ra-ja-se** | thing + ethnic + instrumental |
| **-se-ra-te** | da-**se-ra-te** | instrumental + thing + locative |

**Slot order** (inner → outer):
- Position 1 (stem-adjacent): -ra, -ro, -se, -na, -te, -ne
- Position 2: -ja, -me, -ne, -te, -se
- Position 3 (outermost): -ne, -se

## Reduplication = Plural

34 reduplicated forms found. 5 have attested unreduplicated bases:

| Base | Reduplicated | Document |
|------|-------------|----------|
| a-di-ki-te | a-di-ki-**te-te** | PK Za 11 |
| i-da | i-**da-da** | — |
| ja-sa | ja-**sa-sa**-ra-me | formula |
| ku-pa₃ | ku-**pa₃-pa₃** | — |
| na-ma-ti-ti | na-**ma-ma**-ti-ti-*319 | — |

**Schurr's rule confirmed:** -ja (persons) vs -ra (things) in reduplicated derivatives:
- ki-ro → ki-**ki**-ra-**ja** ("the missing **people**")
- ku-da → ku-**ku**-da-**ra** ("the missing **things**")

## Hurrian Comparison

| LA suffix | Hurrian parallel | Strength |
|-----------|-----------------|----------|
| **-se** | ergative -s(e)/-z(e) | **STRONG** — both mark agent/instrument |
| **-ja** | relational -ya | **STRONG** — both form adjectives |
| **-na** | article -na/-ni | Moderate |
| **-te** | allative -da | Moderate (t/d alternation) |
| **-ne** | genitive -(v)e | Moderate |
| **-me** | 1sg possessive? | Weak (Hurrian 1sg is -tta) |
| **-ro** | (none) | **Unique to Minoan** |
| **-ra** | (none) | **Unique to Minoan** |

Two suffixes (-ro, -ra) have NO parallel in Hurrian or any proposed source language. These are core Minoan morphemes.

## What This Proves

1. **Minoan is exclusively suffixing** — no productive prefixes at all
2. **Minoan is agglutinative** — suffixes stack up to 3 deep in fixed order
3. **Minoan is NOT Greek** — no fusional morphology, no augments, no articles
4. **Minoan is NOT Semitic** — no prefixes, no root-and-pattern
5. **Minoan RESEMBLES Hurrian** typologically — but 2 of 8 core suffixes have no Hurrian parallel
6. **Minoan has a person/thing distinction** in derivation (-ja vs -ra) — rare cross-linguistically
7. **The -ro suffix is unique** — no known parallel in any proposed cognate language
