# Re-discovery of `ku-ro` as a summation marker

## Summary
This benchmark tests whether the current corpus can recover `ku-ro` as an administrative summation marker without simply hardcoding the answer in advance.

## Why it matters
This is a **replication benchmark**. If the project cannot recover known results from corpus structure, then later discovery claims are much weaker.

## Headline result
Using a semi-blind structural method, `ku-ro` emerged as the dominant first token in quantity-bearing tablet lines.

## Status
**provisional success**

## Related findings
- `po-to-ku-ro` appears as a rarer but structurally parallel extended summation expression.
- `ki-ro` appears as a lower-frequency related administrative marker.
