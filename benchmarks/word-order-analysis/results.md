# Results: Word Order Analysis

## Corpus summary
- Documents with full sign sequences: **195** (tablets primarily)
- Sign role categories: Syllabogram, Logogram, Fraction, Transaction sign

## Dominant word order patterns

| Pattern | Count | Example |
|---------|-------|---------|
| WORD | 32 | Names-only tablets/nodules |
| WORD -> IDEO | 20 | Name + commodity |
| WORD -> IDEO -> WORD | 12 | Name + commodity + name |
| WORD -> IDEO -> WORD -> IDEO | 8 | Name + commodity + name + commodity |

## The administrative entry structure

The overwhelming pattern on tablets is:

```
WORD (syllabograms) -> IDEOGRAM (commodity) [-> FRACTION/NUMBER]
```

Repeating for each entry, with a final:
```
ku-ro (total word) -> IDEOGRAM -> FRACTION/NUMBER
```

This confirms: **AGENT/NAME . COMMODITY . QUANTITY** word order.

## Transaction signs

Transaction signs (AB04/te, A307, A516, etc.) appear **before** commodities:

| Transaction sign | Followed by |
|-----------------|-------------|
| AB04 (te) | GRA (3x), VIN (2x), other commodities |
| A307 | VIR (person), syllabograms |
| A516, A326, A319 | Syllabograms (names?) |

This suggests: **TRANSACTION-TYPE . COMMODITY . ENTRIES . TOTAL**

## What precedes logograms?

| Preceding role | Count |
|---------------|-------|
| Logogram | 198 |
| Syllabogram | 141 |
| Fraction | 58 |
| Transaction sign | 18 |

Logograms following syllabograms = NAME + COMMODITY.
Logograms following logograms = COMMODITY LISTS.
Logograms following fractions = QUANTITY + NEXT COMMODITY.

## Verb morphology (M3 #7)

Linear A administrative texts show **no clear verb morphology**. The tablets are noun-heavy lists:
- Names (syllabograms) 
- Commodities (logograms)
- Quantities (fractions/numbers)
- Totals (ku-ro)

This is consistent with **nominal sentence structure** common in ancient Near Eastern accounting:
"X: grain 5, oil 3. Total: 8." — no verbs needed.

**If verbs exist in Linear A, they are likely confined to:**
1. Ritual texts (libation formulas — "I dedicate")
2. Non-administrative inscriptions (not well represented in corpus)
3. The ivory scepter (which may contain narrative text)

## Number marking (M3 #8)

No evidence of grammatical number marking (singular vs. plural) on nouns.
- The same word form appears regardless of quantity
- Commodity ideograms don't change form for different amounts
- This is consistent with the language being **number-neutral** in morphology, like many agglutinative languages

## Confidence
**Provisional success.**
The AGENT-COMMODITY-QUANTITY pattern is robustly attested. The absence of verb morphology is a negative finding that depends on the administrative genre dominating the corpus.
