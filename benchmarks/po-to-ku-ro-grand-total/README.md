# Re-discovery of `po-to-ku-ro` as an extended summation expression

## Summary
This benchmark tests whether the current corpus supports `po-to-ku-ro` as a rarer extended summation expression related to `ku-ro`.

## Why it matters
This is a **replication benchmark**. It checks whether the corpus can recover not just the dominant summation marker, but also a less frequent extended form in the same administrative pattern.

## Headline result
`po-to-ku-ro` appears rarely, but in the current corpus it occurs in the same line-initial quantity-bearing tablet pattern as `ku-ro`.

## Status
**provisional success**

## Related findings
- `ku-ro` is the dominant summation marker in the current administrative corpus.
- `ki-ro` appears as a lower-frequency related administrative marker.
